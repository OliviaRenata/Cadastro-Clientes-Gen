# Cadastro de Clientes - Projeto Genérico

Projeto demonstrativo de cadastro e edição de clientes com PHP.
Todos os dados são fictícios, criados para fins de portfólio.

## Tecnologias
- PHP
- Bootstrap
- JSON para dados de teste

## Como usar
1. Abrir no VS Code Web
2. Rodar com servidor PHP local (ex: Live Server)
3. Visualizar clientes na tabela
